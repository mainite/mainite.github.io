github项目移植到码云

2017-11-22 git

我们都知道github管理代码很方便，但是github毕竟是国外的，
跑起来很慢，并且有大文件限制，为了方便团队开发，有时候
我们会选择码云

下面说一下github里的项目移植码云的方法
首先 查看git库关联的远程代码管理平台

git remote -v

我们看到他指向的是github

我们可以删除关联

git remote rm origin

在关联码云的远程库链接

git remote add origin git@gitee.com:liaoxuefeng/learngit.git

或者直接关联，但是push的时候要用下面选择push

git push github master

或者

git push gitee master

上面的是传到github下面的是传到码云
