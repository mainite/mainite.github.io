Python安装与使用

2017-05-14 Python

首先登陆Python.org下载2.7版本的Python
下载完之后默认安装 路径随便填写

安装完时候关闭

cmd 运行 python 会提示不是内部的命令

这时候我们需要右击我的电脑 点 高级系统设置

点击 环境变量 —> 编辑Path

然后编辑文本 在最后面 加上；然后填写 自己安装的路径
如果所示

点击确定 关闭窗口
cmd 重新输入 python

恭喜 你 你已经完成了python的安装及使用
