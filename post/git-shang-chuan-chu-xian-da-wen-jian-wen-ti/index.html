git上传出现大文件问题

2017-07-08 git

上传github时候出现大文件问题

然后上网查了一堆别人的博客，基本都是说把大文件找出来然后删了，但是笔者想了一下这样的话会出现项目不完整
于是自己查官方然后自己总结出一套解决方案

首先

find ./ -type f -size +102400k

这条命令用于找到自己项目的大文件，

然后登陆https://git-lfs.github.com/

下载官方的工具

按照官方的提示用其管理刚刚找到的大文件

然后 将找出的大文件移到项目文件夹外

其余文件上传成功后

将大文件移会

再次上传即可上传成功
